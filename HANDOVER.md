# 💎 AI 지능형 매매 시스템 운영 및 인계 보고서

## 1. 시스템 현황 (최종 업데이트: 2026-02-24)
본 시스템은 업비트(Upbit)를 포함한 CCXT 기반 거래소에서 10개 주요 코인을 대상으로 작동하는 **실시간 추적형 스캘핑 시스템**입니다.

## 2. 핵심 전략 및 로직 (강화된 하이퍼 스캘핑)
*   **실시간 가격 감시 (Real-time Monitor):** 보유 중인 코인에 대해 **1초 단위 전담 감시 루프**가 작동하여 급변하는 시장 상황에 즉각 반응합니다.
*   **본전 보존 (Break-even):** 수익이 **+0.2%**에 도달하면 손절선을 매수가로 상향하여 리스크를 0으로 만듭니다.
*   **추격 매도 (Trailing Stop):** 수익 **+0.4%**부터 가동되며, 고점 대비 **0.15% 하락** 또는 **RSI 85(과열)** 시 익절합니다.
*   **재매수 쿨다운 (5분):** 매도 후 동일 코인에 대한 휩소(속임수) 방지를 위해 **5분간** 재진입을 금지합니다.
*   **시장 안전 장치:** 비트코인(BTC)의 5분 변동률 및 EMA 정배열 여부를 확인하여 하락장에서는 매수를 중단합니다.

## 3. 서버 운영 가이드 (SSH/Docker)
텔레그램 중복 메시지 및 프로세스 꼬임 방지를 위한 **표준 업데이트 절차**입니다.

```bash
# 1. 프로젝트 폴더 이동
cd ~/coin

# 2. 시스템 완전 종료 및 정리
docker-compose down

# 3. 최신 코드 반영 (Gemini 업데이트 내용)
git pull origin main

# 4. 재빌드 및 무중단 실행 시작
docker-compose up -d --build

# 5. 실시간 로그 감시 (중상 여부 확인)
docker-compose logs -f --tail=50
```

## 4. 예상 매매 성과 (통계 모델)
*   **예상 승률:** 65% ~ 75% (본전 보존 로직 덕분)
*   **손익비:** 익절(+0.4%~0.8%) vs 손절(-0.3%)
*   **일일 평균 거래:** 15회 ~ 30회 (시장 상황에 따라 변동)

## 5. 보안 및 관리 주의사항
*   **환경 변수:** `.env` 파일에 API 키와 텔레그램 정보가 안전하게 관리되어야 합니다.
*   **최소 주문액:** 업비트 기준 5,000원 이하 주문은 자동으로 필터링됩니다.
*   **중복 실행 금지:** 텔레그램 메시지 중복 오류 발생 시 반드시 `docker-compose down` 후 재시작하십시오.
